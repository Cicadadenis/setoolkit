RATTE (инструмент удаленного администрирования Tommy Edition) - это полезный модуль и сторонний модуль, добавленный / созданный в SET с помощью
Томас Верт.

Что касается RATTE, то следует отметить, что его основная цель и дизайн - полностью обойти ограничения на основе выходного кода и брандмауэра, используя чисто HTTP-коммуникации для команд туда и обратно.
RATTE был расширен, чтобы быть очень настраиваемым.
На данный момент вы можете установить:
- Подключи Back IP
- Подключите задний порт
- Wheater RATTE является постоянным или нет (пример: для тестирования сетевого брандмауэра нет необходимости быть постоянным. В тесте на проникновение все может выглядеть иначе)
- пользовательское имя файла, которое RATTE использует для запуска, так что локальные брандмауэры и пользователь могут быть одурачены, используя такие имена, как iexplore.exe или firefox.exe и т. д.

Если RATTE является постоянным, он пытается в системах NTFS внедрить себя в двоичный файл браузера по умолчанию и заменяет
исполняемые файлы с частью собственного кода. В случае неудачи RATTE сохранит себя как приложение автозапуска, используя указанное имя файла. Если этот файл отсутствует, он будет выглядеть как iexplore.exe.

RATTE использует связь с microsoft.com для определения выхода из сети.
